# API Testing Guide

## Quick Testing with cURL

### 1. Health Check
```bash
curl http://localhost:5000/api/health
```

Expected Response:
```json
{"message":"Server is running"}
```

### 2. Upload Image
```bash
curl -X POST http://localhost:5000/api/images/upload \
  -F "image=@test-image.jpg"
```

Expected Response:
```json
{
  "message": "Image uploaded successfully",
  "image": {
    "id": "507f1f77bcf86cd799439011",
    "filename": "image-1716792600000-456789.jpg",
    "originalName": "test-image.jpg",
    "filePath": "/uploads/image-1716792600000-456789.jpg",
    "size": 45678,
    "uploadedAt": "2026-05-27T05:56:40.123Z"

}
```

## Testing with Postman

### Setup

1. **Create Collection**: "MERN Image Upload"
2. **Set Base URL**: `{{base_url}}` = `http://localhost:5000`

### Requests

#### POST - Upload Image
- **URL**: `{{base_url}}/api/images/upload`
- **Method**: POST
- **Body**: form-data
  - Key: `image` (type: File)
  - Value: Select your image file
- **Headers**: Auto-generated by Postman

#### GET - All Images
- **URL**: `{{base_url}}/api/images`
- **Method**: GET

#### GET - Single Image
- **URL**: `{{base_url}}/api/images/:id`
- **Method**: GET
- **Params**: id = (paste your image ID)

#### DELETE - Delete Image
- **URL**: `{{base_url}}/api/images/:id`
- **Method**: DELETE
- **Params**: id = (paste your image ID)

## Error Test Cases

### Invalid File Type
```bash
curl -X POST http://localhost:5000/api/images/upload \
  -F "image=@document.pdf"
```

Response: 400 Bad Request
```json
{"error":"Only image files are allowed"}
```

### Missing File
```bash
curl -X POST http://localhost:5000/api/images/upload
```

Response: 400 Bad Request
```json
{"error":"No file uploaded"}
```

### File Too Large (>5MB)
```bash
curl -X POST http://localhost:5000/api/images/upload \
  -F "image=@large-file.jpg"
```

Response: 413 Payload Too Large
```json
{"error":"Failed to upload image"}
```

### Invalid Image ID
```bash
curl http://localhost:5000/api/images/invalidid123
```

Response: 404 Not Found
```json
{"error":"Image not found"}
```

## Performance Metrics

Test upload with various file sizes:
- Small (100KB): < 100ms
- Medium (1MB): < 500ms
- Large (5MB): < 2000ms
